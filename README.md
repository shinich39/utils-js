# utils-js

Utilities for javascript development.

## Usage

ESM

```js
import * as utils from "./dist/utils.min.mjs";
```

CJS

```js
const utils = require("./dist/utils.min.cjs");
```

Browser

```html
<script src="./dist/utils.min.js"></script>
<script>
  const { isBoolean, } = window.utils;
</script>
```
